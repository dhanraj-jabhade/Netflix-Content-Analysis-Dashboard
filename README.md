# Netflix Content Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard provides insights into Netflix content by analyzing movies, TV shows, genres, ratings, countries, actors, directors, and content growth trends.

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## 📊 Key Metrics
- Total Content: 7K+
- Movies: 4K+
- TV Shows: 3K+
- Countries: 58

## 📈 Dashboard Insights
- Top Genres on Netflix
- Content Rating Distribution
- Movies vs TV Shows Analysis
- Production Country Analysis
- Top Actors by Appearance
- Top Directors
- Content Growth Over Time
